# Engineering reference — Calibre 44

The operator's Calibre 44 unibody automatic movement package. Kept as the **exemplar of the
build discipline**, not a blueprint to copy — "the design may need work" (operator). What ports
is the *philosophy*: constancy, serviceability, machine-honest finish, scope discipline (every
omitted complication is a part that cannot fail), a bounded counted manifest, the mainspring at
center. See docs/V2_DEFINITION.md §0.5.
